# Profiling &amp; Optimisation workshop for Python &amp; R

This repository contains sample code for the [workshop at Imperial College’s School of Public Health](https://sangeetabhatia03.github.io/green-computing-workshop/).

## Additional Resources

* For Python: [Profiling and Optimisation course](https://carpentries-incubator.github.io/pando-python/)
* For R: [Advanced R book](https://adv-r.hadley.nz/techniques.html)
* [Reasonable Performance Computing SIG website](https://sig-rpc.github.io), which includes databases of profiling tools and optimisation techniques for many different programming languages
